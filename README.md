# 🚀 CodeMentor AI – Coding Interview Preparation System

A beginner-friendly Java-based application designed to help students prepare for coding interviews through skill tracking and simple user management features.

- User Registration
- Skill Tracking
- Simple Dashboard UI
- Beginner-Friendly Interface
- Java-Based Application
